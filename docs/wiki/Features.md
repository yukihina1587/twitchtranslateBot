# 機能一覧

## チャット翻訳

### 自動翻訳

Twitchチャットのメッセージを自動的に翻訳します。

- **翻訳モード**:
  - 自動: 日本語→英語、その他→日本語
  - 英→日: 英語を日本語に翻訳
  - 日→英: 日本語を英語に翻訳

### 翻訳フィルター

特定のワードを含むメッセージの翻訳をスキップします。

- 設定パネルの「辞書」タブで設定
- 部分一致で判定

### カスタム辞書

翻訳前にテキストを置換します。

- 固有名詞の翻訳を制御
- 例: `YAGOO` → `谷郷`

---

## 音声機能

### チャット読み上げ（TTS）

日本語のチャットメッセージを音声で読み上げます。

| エンジン | 品質 | 必要なもの |
|---------|------|-----------|
| VOICEVOX | 高 | VOICEVOX Engine |
| pyttsx3 | 低 | なし（フォールバック） |

**ボイス選択**: 設定パネルで複数のボイスから選択可能

### 読み上げ辞書

漢字の読み間違いを修正します。

- 例: `草` → `くさ`（「そう」ではなく）
- 設定パネルの「辞書」タブで設定

### 音声翻訳（マイク入力）

マイクに向かって喋った内容を翻訳します。

| プロバイダ | 品質 | 制限 |
|-----------|------|------|
| Gladia API | 高 | 月10時間（無料枠） |
| Google SR | 中 | 制限なし（フォールバック） |

## Twitchイベント検知

### 対応イベント

| イベント | 検知方法 | 効果音 |
|---------|---------|--------|
| フォロー | EventSub WebSocket | ✅ |
| サブスク | IRC usernotice | ✅ |
| ギフトサブ | IRC usernotice | ✅ |
| Bits | IRC usernotice | ✅ |

### イベント効果音

各イベントに個別の効果音を設定できます。

- 対応形式: MP3, WAV, OGG
- 個別のボリューム調整
- 左サイドバーでテスト再生

---

## 参加者追跡

特定のキーワードを含むメッセージを送信したユーザーを追跡します。

- キーワード設定可能（例: `参加`, `join`）
- 参加者リストをチャットに送出可能
- 設定パネルの「参加者」タブで管理

---

## オーバーレイ

OBSなどの配信ソフトで使用できる翻訳オーバーレイを提供します。

- **URL**: `http://localhost:8080/overlay.html`
- リアルタイム更新
- 詳細は[API参照](API-Reference)を参照

## UIテーマ

4種類のテーマから選択できます。

| テーマ | 説明 |
|-------|------|
| デフォルト | クラシックなダークテーマ |
| グラデーション | モダンなグラスモーフィズム |
| ミニマル | シンプルなライトテーマ |

## ログ機能

### ログファイル

- 場所: `dist/logs/bot_YYYY-MM-DD.log`
- 日付ごとにローテーション
- 7日間保持

### ログレベル

GUIから動的に変更可能:

- DEBUG: 詳細なデバッグ情報
- INFO: 通常の動作情報
- WARNING: 警告
- ERROR: エラーのみ
